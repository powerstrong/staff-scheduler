# staff-scheduler

근무기간, 근무자, 휴무 규칙을 입력하면 조건에 맞춰 근무자를 자동 배치하는 정적 웹앱입니다.

## 구성

- `index.html`, `styles.css`, `script.js`: 웹앱 본체
- `.github/workflows/deploy-pages.yml`: GitHub Pages 배포 워크플로

## 주요 기능

- 배치 우선순위 직접 입력
- 주간 자동 휴무 요일 직접 선택
- 월별 자동 휴무 규칙 직접 입력
- 커스텀 휴무일 / 무료휴일 / 커스텀 근무일 설정
- `텍스트로 보기`, `시각화해서 보기` 탭 제공
- 반년치 일정도 한 화면에서 보기 쉽도록 압축된 캘린더 표시

## 사용 방법

1. 시작일과 종료일을 입력합니다.
2. 근무 인원을 입력합니다.
3. 배치 우선순위와 자동 휴무 규칙을 설정합니다.
4. 필요하면 커스텀 휴무일과 커스텀 근무일을 추가합니다.
5. 스케줄 생성 버튼을 누릅니다.

## 배포

이 저장소는 GitHub Actions로 GitHub Pages에 배포되도록 설정되어 있습니다.
